# default

這是一個用 Vue3, Vuetify 的專案，列出世界上所有國家的資訊，並且可以透過搜尋功能找到特定國家的資訊。

## Demo

https://vuetify-countries.netlify.app/

## Project setup

```
# yarn
yarn
yarn dev
```
